# Callout - Guidelines

## Guidelines

Inline contextual highlight — lighter than [[Banner]], more prominent than body text. Used to draw attention to key information within a content section.

### Imports

**Web**:
```tsx
import { Callout } from '@shinetools/lumen-react';
```

**Mobile**:
```tsx
import { Callout } from '@shinetools/lumen-native';
```

### Basic Usage

```tsx
<Callout />
```
