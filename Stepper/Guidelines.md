# Stepper - Guidelines

## Guidelines

Guides users through a multi-step sequential process. Shows progress and allows navigation between completed steps.

### Imports

**Web**:
```tsx
import { Stepper } from '@shinetools/lumen-react';
```

**Mobile**:
```tsx
import { Stepper } from '@shinetools/lumen-native';
```

### Basic Usage

```tsx
<Stepper />
```
