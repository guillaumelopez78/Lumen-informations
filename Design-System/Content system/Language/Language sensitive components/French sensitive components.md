# French sensitive components

## 🎼 Informal tone vs. Formal tone

- Always use second person plural (”vous”) `Votre entreprise` `Ton entreprise`
- Use polite formulations `Veuillez saisir…` `Merci de…` `Nous vous invitons à…`
- Maintain a respectful distance whilst being warm and supportive `Vérifiez vos informations bancaires et réessayez.`
- Avoid overly casual expressions or slang `Oups ! Ton paiement n'a pas marché.` `Content de vous voir ici.`

## ⚧️  Gender-neutral language

#### IF a neutral option is possible:

- Use gender-neutral alternatives for nouns `L'équipe dirigeante` `Les Directeurs`
- Include both gender for demonstrative pronouns `Celles et ceux` `Toutes et tous` `Ceux·lles` `Tous·tes`
- Whenever possible, reformulate to avoid gendered nouns entirely `Il est possible de modifier son profil` `Toute la clientèle` `Les utilisateurs peuvent modifier leur profil` `Tous les clients`

#### IF a neutral option is impossible:

**Use midpoint (point médian** `Shift + option + F`**) sparingly** to create gender-inclusive forms

- Use **only one midpoint per word**
- Place it between the masculine base and the first different letter of the feminine form `Nous sommes heureux·ses` `Nous sommes heureux·se·s`

| ❌ Masculine default | ✅ Inclusive (with midpoint) | ✅ Gender-neutral alternative |
| --- | --- | --- |
| Les utilisateurs | Les utilisateur·ices | Les personnes qui utilisent, La communauté |
| Les dirigeants | Les dirigeant·es | Les membres de la direction, L'équipe dirigeante |
| Les indépendants | Les indépendant·es | Le personnel, L'équipe, Les collaborateur·ices |
| Directeur | Directeur·ice | Bonjour, Madame, Monsieur (if formal context requires) |

## 🇬🇧 Anglicism

Anglicism are English words used in another language. They are allowed only when no French word is obvious enough.

| **✅ To favour** | **🚫 To avoid** | **Why ?** |
| --- | --- | --- |
| **API, SaaS, cloud** | **Features, onboarding, roadmap** | Those words are widely accepted in French. |
| **Interface, tableau de bord** | **Dashboard** | « Tableau de bord » is more meaningful. |
| **Compte, espace personnel** | **Account, login** | French words guarantees a better accessibility and understanding. |
| **Basé sur l’intelligence artificielle** | **AI-driven** | Too abstract for the target audience. |

## 🗓️ Numbers and dates

- Months and days of the week use lowercase unless sentence-initial `janvier` `lundi` `jan.`
- Thousands separator are space or full stop, decimal use a comma `1 250,50` `1,250.0`
- "Premier" is written as "1er", all others use "e” `1er/1re, 2e, 3e, 4e…`
- Phone numbers use country code and spaces to group digits `+33 1 23 45 67 89`

## ⚛️ Other sensitive components

- Add a non-breaking space *before* `: ; ! ? " "` never before `.` or `,`
- Titles use the polite form (vouvoiement), not the infinitive `Créez un client` `Créer un client`
