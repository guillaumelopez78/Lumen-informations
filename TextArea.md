## Guidelines

Multi-line text input. Use when free-form text is expected to span multiple lines (notes, descriptions, messages).

### Imports

**Web**:
```tsx
import { Text Area } from '@shinetools/lumen-react';
```

**Mobile**:
```tsx
import { Text Area } from '@shinetools/lumen-native';
```

### Basic Usage

```tsx
<Text Area />
```
