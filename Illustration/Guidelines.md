# Illustration - Guidelines

## Guidelines

Lumen's illustration set for empty states, onboarding, error pages, and confirmations.

### Imports

**Web**:
```tsx
import { Illustration } from '@shinetools/lumen-react';
```

**Mobile**:
```tsx
import { Illustration } from '@shinetools/lumen-native';
```

### Basic Usage

```tsx
<Illustration />
```
