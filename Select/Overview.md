# Select

## Purpose

Dropdown menu for choosing one option from a list of 5+ items.

## When to use

Use select for:

- Country or region selection
- Time zone and language choosers
- Category filtering
- Status or priority selection

## When not to use

- If a simpler pattern exists, prefer it
- For accessibility-critical flows, always test with screen readers
- If the action requires context, use a Dialog instead
- For instant-effect settings, use Switch (not Checkbox)
