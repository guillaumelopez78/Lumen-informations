# Inputs

## How to write inputs:

#### IF input has a label:

- Do not use verbs, only nouns `Company logo` `Upload your logo`

#### IF input has a placeholder in field:

- Placeholders do not repeat the label and are not a description, it shouldn’t explain the expected action but rather indicate a pattern `+33 6 00 00 00 00` `Enter your phone number here.`

#### IF input has a hint outside the field:

- Add complementary information or instruction on format
- Go straight to the point `GIF, JPG or PNG under 2Mo.`
- Don't use hint if nothing has to be added
- ⚠️ **Beware of the length: always test it on mobile design**

#### IF it’s a search input:

- Put the keywords users can search for `Search by name, invoice number, client…`
- Never put `Search` alone
- If you have +1 example in the list and list is not complete, use ellipsis
- ⚠️ **Beware of the length: always test it on mobile design**
