## Guidelines

Allows users to attach files. Supports drag-and-drop and click-to-browse.

### Imports

**Web**:
```tsx
import { File Upload } from '@shinetools/lumen-react';
```

**Mobile**:
```tsx
import { File Upload } from '@shinetools/lumen-native';
```

### Basic Usage

```tsx
<File Upload />
```
