## Guidelines

Dropdown input for selecting multiple options from a list. Selected items appear as tags inside the trigger.

### Imports

**Web**:
```tsx
import { Multi Select } from '@shinetools/lumen-react';
```

**Mobile**:
```tsx
import { Multi Select } from '@shinetools/lumen-native';
```

### Basic Usage

```tsx
<Multi Select />
```
