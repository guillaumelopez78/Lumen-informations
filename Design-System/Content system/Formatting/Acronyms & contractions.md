# Acronyms & contractions

## How to write acronyms:

- Always write acronyms in all caps, no points: **`API`** **`VAT`** **`PDF`** — never **`A.P.I**.` or **`V.A.T.`**
- Spell out the full term on first use, with the acronym in parentheses: **`Value Added Tax (VAT)`,** then you can use the acronym alone
- Never use acronyms that aren't standard or widely known — write the full term instead

#### Common acronyms

**`IBAN`**   **`VAT`**   **`API`**   **`PDF`**   **`JPG`**   **`PNG`**   **`CSV`**   **`EU`**   **`US`**

## How to write contractions:

- Always use a full stop: **`etc.`**   **`e.g.`**   **`i.e.`**   **`approx.`   `max.`   `min.`**
- **`etc.`** is always followed by a full stop, never use **`etc...`**
