## Guidelines

Input for selecting a single date via a calendar overlay. For date ranges, see [[Period-Picker]].

### Imports

**Web**:
```tsx
import { Date Picker } from '@shinetools/lumen-react';
```

**Mobile**:
```tsx
import { Date Picker } from '@shinetools/lumen-native';
```

### Basic Usage

```tsx
<Date Picker />
```
