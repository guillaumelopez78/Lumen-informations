# TextField

## Purpose

Single-line text input for user data entry with validation and help text.

## When to use

Use textfield for:

- Email, phone, and username inputs
- Search queries with placeholders
- Real-time form validation
- Numeric and password inputs with masking

## When not to use

- If a simpler pattern exists, prefer it
- For accessibility-critical flows, always test with screen readers
- If the action requires context, use a Dialog instead
- For instant-effect settings, use Switch (not Checkbox)
