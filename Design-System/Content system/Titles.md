## How to write H1 titles:

#### IF title is for a verification screen

Verification screen doesn’t have an instruction and help to consult something specific related to the user themself.

- Use: “Your” + the topic of the screen `Your company` `Your subscription`

#### IF title is for an information screen

Information screen doesn’t have an instruction. An information title always has a subtitle.

- Use the name of the feature or the specific name of the information `Dashboard`

#### IF title is for an instruction screen

Instruction screen guides users through a flow.

- Action verb + the action needed from the user: `Specify your role in the company`
- Always stay affirmative, never use questions: `What’s your role in the company?`

<aside>
🇫🇷

In French, the verb is conjugated using the “vouvoiement” (polite form) `-ez`, not infinitive `-er`.

</aside>

## How to write H2/H3 titles:

- Be specific on the section's **content**: `Payment detail`  `Personal information`
- **Don't repeat the H1**, if no new information is to be displayed, remove the H2: 
**`No data** 
There is no data yet.`

## How to write subtitles:

- **Don't repeat the H1 or** **H2**, if no new information is to be displayed, remove the subtitle
- Focus on the **benefits** the user will get: `Estimate your company’s social contributions and anticipate your due dates`

<aside>
💡

**Reminders:**

- Write in the active voice and avoid passive grammatical constructions.
- Always try to frame things in terms of what's possible and available rather than what's missing, problematic, or just plain negative.
- Use “we” when talking about collective effort, use “you” when referring to a user’s achievements.
</aside>
