# Capitalisation

## Core rules

- Use sentence case everywhere: only the **`first word`** and **`proper nouns`** get a capital
- Never use all-caps for emphasis — use **bold** or *italics* instead
- Months and days follow language-specific conventions — see market rules below

**`Create a new payment`** , **`Export transaction history`**
**`Create a New Payment Export`** , **`Transaction History`**

## Quick reference

| Element | Capitalisation | Example |
| --- | --- | --- |
| Headings | Sentence case | **`Manage your payments`** |
| Buttons | Sentence case | **`Add new card`** |
| Form labels | Sentence case | **`E-mail address`** |
| Error messages | Sentence case | **`Payment method not supported`** |
| Acronyms | All caps | **`API`**, **`VAT`**, **`KYC`** |
| Swift | Initial cap only | **`Swift code`** |
| Brand names | Follow official | **`PayPal`**, **`Mastercard`**, **`iOS`** |

When in doubt, use sentence case. It's friendlier, more readable, and aligns with modern interface design principles.

## Months & days – market-specific rules

<aside>

**English (default)**

Always capitalise **`January`** **`Monday`** **`Jan`**

Lowercase unless sentence-initial **`janvier` `lundi` `jan.`**

</aside>

Lowercase unless sentence-initial **`januar`** **`mandag`** **`jan`**

## Go further

<aside>
💡

### **Why sentence case?**

Sentence case is more conversational and readable than title case. It's the standard for modern product interfaces and aligns with how users naturally read.

### **Swift — a common mistake**

The organisation rebranded and now uses "Swift" with an initial cap only — not "SWIFT".

**`Swift payment`**   **`Swift code`**   **`Swift network`**   **`SWIFT payment`**   **`SWIFT code`**

### **Other frequently miscapitalised terms**

- **`API key`**   **`API Key`**
- **`End-to-end encryption`**    **`End-To-End Encryption`**
- **`Two-factor authentication`**    **`Two-Factor Authentication`**
- **`internet` `website` `e-mail` `online`**

### **After colons**

Use lowercase after a colon unless what follows is a complete sentence or proper noun.

**`Payment status: pending`**   **`Note: The transaction will be processed within 24 hours.`**

### **Hyphenated words**

Only capitalise the first element, unless the second is a proper noun.

**`Cross-border payments`**   **`UK-based merchant`   `Cross-Border Payments`**

### **Numbers starting a sentence**

Never start a sentence with a numeral — write it out or restructure.

**`Three payment methods are available`**   **`There are 3 payment methods available`**   
**`3 payment methods are available`**

</aside>
