# Inclusive language

<aside>

Inclusive language ensures all users feel respected and represented in our product. We use language that avoids assumptions about gender, ability, background, or identity, creating a welcoming experience for everyone.

</aside>

<aside>
⚠️

**IMPORTANT: Use inclusive writing only when a neutral/non-gendered alternative is impossible.**

</aside>

## ⚧️ Gender-neutral language

### 🇬🇧 **English**

Use gender-neutral terms by default:

| ❌ Gendered | ✅ Gender-neutral |
| --- | --- |
| Chairman | Chair, Chairperson |
| Businessman | Business owner, Entrepreneur |
| Salesman | Salesperson, Sales representative |

Avoid gendered pronouns when gender is unknown or irrelevant:

| ✅ Gender-neutral |
| --- |
| The user can update their profile |
| Each customer should verify their details |
| Ask your accountant for their advice |

<aside>
🎓

**Note:** Singular "they/their" is grammatically correct and widely accepted in modern English.

</aside>

### 🇫🇷 **French**

**French inclusive writing is particularly important due to grammatical gender.** Our approach balances inclusivity with readability.

**Use midpoint (point médian** `Shift + option + F`**) sparingly** to create gender-inclusive forms:

| ❌ Masculine default | ✅ Inclusive (with midpoint) |
| --- | --- |
| Les utilisateurs | Les utilisateur·ices |
| Les dirigeants | Les dirigeant·es |
| Les indépendants | Les indépendant·es |
| Directeur | Directeur·ice |

Guidelines for midpoint usage:

- Use **only one midpoint per word**
- Place it between the masculine base and the first different letter of the feminine form
- Example: "utilisateur·ices" (not "utilisateur·trice·s")

Prefer gender-neutral alternatives when available:

| ❌ Gendered | ✅ Gender-neutral alternative |
| --- | --- |
| Les utilisateurs | Les personnes qui utilisent, La communauté |
| Les directeurs | Les membres de la direction, L'équipe dirigeante |
| Les employés | Le personnel, L'équipe, Les collaborateur·ice·s |
| Cher client | Bonjour, Madame, Monsieur (if formal context requires) |

**Collective terms and reformulation:**

Whenever possible, reformulate to avoid gendered nouns entirely:

| ❌ Gendered | ✅ Reformulated (neutral) |
| --- | --- |
| Les utilisateurs peuvent modifier leur profil | Il est possible de modifier son profil |
| Tous les clients | Toute la clientèle |
| Les entrepreneurs | Les personnes qui entreprennent |

### **Examples of inclusive vs non-inclusive**

✅ **Correct usage (inclusive):**

- "Vos associé·e·s ; les dirigeant·e·s ; les indépendant·e·s ; nous sommes heureux·se·s ; utilisateur·ice·s ; directeur·ice·s"

❌ **Incorrect usage (avoid):**

- "Vos associé·e·s ; les dirigeant·e·s ; les indépendant·e·s ; nous sommes heureux·se·s ; utilisateur·ice·s ; directeur·trice·s ; lels ; celleux"
- Over-use of midpoints makes text difficult to read and can reduce accessibility for screen readers

**Warning (Épicène):**
Use inclusive writing **only when a neutral/non-gendered option is impossible.** Prefer alternatives like:

- "celles et ceux" (those)
- "ils ou elles" (they - masculine or feminine)
- "toutes et tous" (everyone)

Avoid making text **illegible or overly complex** by over-using inclusive forms.
