## Guidelines

Displays an already-generated or uploaded file as a downloadable/viewable item. Typically appears in transaction details, invoice previews, and document lists.

### Imports

**Web**:
```tsx
import { File Output } from '@shinetools/lumen-react';
```

**Mobile**:
```tsx
import { File Output } from '@shinetools/lumen-native';
```

### Basic Usage

```tsx
<File Output />
```
