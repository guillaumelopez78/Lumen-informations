# Notifications

## How to write notifications:

- 

## 🔑 Key points

> **Clarity**
Be short and concise, explain the purpose of the notification.
> 

> **Suitability**
Bring something meaningful to the user. Do not interrupt them pointlessly.
> 

> **Customization**
Adapt the message to the context to add relevance.
> 

> **Actionable**
Suggest an concise action (*Example: “Open the app”, “See message”*).
